# Carpeta de productos

Usa esta carpeta para organizar los materiales de producto en subcarpetas por categoría (por ejemplo, `Productos/ropa`, `Productos/electronica`). GitHub creará automáticamente las rutas si subes archivos con el nombre de carpeta incluido.

## Recomendaciones de nombres
- Usa solo letras, números y guiones (`-`) para evitar errores en GitHub.
- Incluye la categoría y el tipo de archivo en la ruta. Ejemplo: `Productos/ropa/camisa-manga-larga.png`.
- Si subes un README u hoja de datos para el producto, usa `README.md` o `descripcion.md` dentro de su carpeta.

Si necesitas versionar una carpeta vacía, añade un archivo `.gitkeep` dentro para que Git la incluya en el repositorio.
